Introdução ao HTML

(Voltar)

Oque é HTML?

A "HyperText Markup Language", comumente conhecida como HTML, é uma linguagem usada para construir páginas da web
O HTML desempenha um papel crucial no estabelecimento da estrutura e do sistema de um site, permitindo a inserção e organização do conteúdo.
Sem HTML, os navegadores teriam dificuldade para exibir elementos como texto e carregar vários tipos de conteúdo, incluindo imagens.
É importante notar que o HTML constitui a base da Internet e é frequentemente combinado com outras linguagens, como CSS e JavaScript, para criar páginas da web dinâmicas e envolventes.

Quem criou o HTML?

O HTML foi criado em 1991 por Tim Berners-Lee, no CERN(Conselho Europeu para a Pesquisa Nuclear) na Suíça.
Feito apenas para instituições de pesquisa e compartilhar documentos com facilidade.

Comunicação Client X Server

(Voltar)

O que são Servers?

Resumindo, um Server(Servidor Web) é um computador que cuida do armazenamento, processamento,
e finalmente a entrega dos arquivos que o site precisa, é assim que entramos em um site, ou recarregamos a página.

O que são Clients?

O Client é o consumidor da informação da internet, ou seja, quando você entra em algum site, O Client(Você) vai encontrar o Server(Computador), que vai enviar o arquivo do site que você precisa para enfim entrar nele.

Como o Client se conecta ao Server?

Explicando de uma forma mais detalhada, quando você hospeda uma site, você envia os arquivos do site para um Servidor,
e quando alguém tentar entrar no seu site, o server envia os arquivos do seu site para o Client, que é a pessoa que vai entrar no seu site.
Após o arquivo ter sido enviado, a pessoa poderá ver e interagir com o seu site.

Entendendo as Tags

(Voltar)

O que são Tags?

uma forma de se comunicar com seu computador e dizer oque ele deve fazer, como um parágrafo, que é representado pelo "< p >"
Aqui estão alguns exemplos de Tags.

Estrutura Básica de um HTML

Todo HTML sempre ira começar com "html", "head" e "body", que é a base de tudo.

A tag "html" é uma das tags mais importantes em HTML. Ela é usada para definir o início do documento HTML e deve ser fechada com a tag "/html".
A tag "head" é um elemento que fornece informações gerais (metadados) sobre o documento, incluindo seu título e links.
A tag "body" é usada para definir o corpo de um site. Tudo o que é visível para o usuário, como textos, links e imagens, estará dentro da tag.

Tags relacionadas a Textos

Agora iremos para alguns comandos relacionados a texto, que nos permite criar, editar a cor, fonte, localização e etc...

A Tag "p" é a principal tag de texto, que compõe um parágrafo.

A Tag "i" é semelhante ao "p", porém ela transforma o texto em Itálico.
A Tag "b" também funciona desse jeito, e transforma o texto em negrito.

Listas Ordenadas e Não Ordenadas

(Voltar)

Listas Ordenadas

(Voltar)

A tag < ol > é uma tag HTML que representa uma lista ordenada. Essa tag é usada para criar listas de itens que precisam ser numerados em uma ordem específica.
Cada item da lista é representado pela tag < li >. A tag < ol > tem vários atributos que podem ser usados para personalizar a aparência da lista,
como start, type e reversed.

Aqui está um exemplo:

 
< ol >
< li >Item 1< / li >
 < li >Item 2< / li >
  < li >Item 3< / li >
< / ol >
  1. item 1
  2. item 2
  3. item 3

Listas Não Ordenadas

(Voltar)

As listas não ordenadas são um recurso HTML que permite criar listas com marcadores. As listas não ordenadas são criadas usando as tags < ul > e < li >.
A tag < ul > é usada para criar a lista não ordenada e a tag < li > é usada para criar cada item da lista. Para criar uma lista não ordenada,
basta digitar < ul > e pressionar a tecla Tab. Em seguida, digite < li > e pressione a tecla Tab novamente para criar um item da lista.
Repita esse processo para adicionar mais itens à lista.

Qual a diferença da Lista Ordenada e da Não Ordenada?

A principal diferença entre uma lista ordenada e uma lista não ordenada é que a lista ordenada é uma lista de itens em que a ordem dos itens é importante, enquanto a lista não ordenada é uma lista de itens em que a ordem dos itens não é importante.

Em uma lista ordenada, cada item é numerado em uma ordem específica, geralmente começando com o número 1.
Já em uma lista não ordenada, cada item é marcado com um ponto, para indicar que os itens não têm uma ordem específica.

Aprendendo a usar Links

(Voltar)

O que são Links?

Um link(ligação) é um endereço de um site, todos os sites tem um link próprio, como por exemplo,
a plataforma de compartilhamento de vídeos "Youtube", cujo link é "https//www.youtube.com"
A rede social "Instagram" tem o endereço "https://www.instagram.com"

Como se usar ele no VSCode?

Para se usar um site, você precisa usar a tag < a >, abrir a tag, e colocar "href", tornando assim "< a href="">"
Entre essas Aspas, coloque o endereço do site, exemplo: < a href="https//www.youtube.com">
por último, após esse comando, antes do fechamento da Tag "a" coloque o nome do site do endereço, que seria "Youtube"(do meu exemplo).

como fazer o link abrir em outra aba?

Você precisara usar o atributo "target" após o link, exemplo: < a href="https//www.youtube.com" target="" >
ainda não está feito, pois após inserir o atributo, você precisa escrever entre as aspas "_blank"
e pronto, você conseguiu criar um texto com um link inserido.
